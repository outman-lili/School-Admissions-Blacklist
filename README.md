# 校招污点名单

本账号及仓库专用于记录在校园招聘中的具有**污点行为**的公司。

污点行为**包括**但不限于：毁意向书、毁两方协定、毁三方协定、试用期裁员、大量裁应届生。

污点行为**不**包括：毁OC（除非大量）、试用期结束未通过（除非大批量蓄意）。

## 项目背景

近年来，无论是校招生的毁约、变相劝退、卡试用期、裁应届，都屡见不鲜。在劳动关系中，雇员本就是弱势的一方，对于没有经验的校招生来说，更是如此。

毁校招生的offer几乎无需赔偿，应届裁员也成本很低。对于雇主来说，或许砍掉应届生是最有性价比的选择。这里我们不探讨毁约/裁员应届生的正确与否，立场不同，无可厚非。

但身为弱势的学生群体，我们总要有起码的知情权，不该今年资本家，明年马爸爸。每个人都该为自己的选择承担代价。

往者不可谏，来者犹可追。希望本名单能够为未来的应届同学选择工作提供参考，也希望能够让雇主对应届生下手时多一点顾虑。

**互联网不该没有记忆！**

- 同公司不同部门情况有可能不同，公司在名单中不代表该公司全部岗位存在该情况，仅供参考。
